## Project Overview

This is a professional, modern website redesign for the New Hampshire Troopers Association, featuring a clean, responsive design with a unified layout system across all pages.

## Features

- **5 Complete Pages**: Home, About Us, Membership, Events, and Contact
- **Unified Layout System**: Shared navigation and footer loaded via JavaScript
- **Modern Design**: Professional color scheme inspired by law enforcement themes
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Smooth Animations**: Fade-in effects and hover interactions
- **Professional Typography**: Clean, readable fonts with proper hierarchy
- **Accessible**: Semantic HTML with proper heading structure

## Technology Stack

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Modern styling with CSS Grid, Flexbox, and CSS Variables
- **Vanilla JavaScript**: No frameworks - clean, efficient code

## How to Use

1. **Open the website**: Simply open `index.html` in any modern web browser
2. **Navigate**: Use the top navigation menu to move between pages
3. **Interactive Elements**: 
   - Hover over cards and buttons for animations
   - Submit the contact form (displays alert - demo only)
   - Click navigation links for smooth page transitions
